We've reorganized the Unkey dashboard so your projects, keyspaces, and ratelimits all live in one place. We've also renamed APIs to Keyspaces (APIs) to better describe what they are: a container for your keys.

![The new Projects-first navigation](/images/changelog/2026-06-01/projects-navigation.png)

## APIs are now Keyspaces (APIs)

You'll find your APIs under **Keyspaces (APIs)** in the sidebar.

Your API ID hasn't changed. It's still `api_…`, it's still what you pass in API calls, and you'll find it next to the KeySpace ID on each keyspace's settings page.

Read more in the [Keyspaces documentation](https://www.unkey.com/docs/platform/apis/overview).
